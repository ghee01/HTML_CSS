# 대구광역시 미세먼지 현황 대시보드

공공데이터포털(data.go.kr) 에어코리아 대기질 정보 API를 기반으로 수집한 데이터를 HTML/CSS와 Chart.js를 활용하여 시각화한 웹 대시보드입니다.

---

## 사용 데이터

| 항목 | 내용 |
|---|---|
| API명 | 한국환경공단 에어코리아 대기질 정보 |
| 출처 URL | https://data.go.kr/tcs/dss/selectApiDataDetailView.do?publicDataPk=15073861 |
| 측정 항목 | PM10 (미세먼지), PM2.5 (초미세먼지) |
| 측정 지역 | 대구광역시 8개 구·군 |
| 기준일 | 2024-06-18 |

> ※ 본 프로젝트는 API 샘플 데이터를 하드코딩하여 사용하였습니다.

---

## 구현 기능

- **HTML 시맨틱 구조** : `header`, `main`, `section`, `footer` 태그로 페이지 구역 구분
- **핵심 통계 표시** : PM10 평균·최대·최소, PM2.5 평균을 카드 형태로 표시 (JavaScript로 계산)
- **막대 차트** : 대구 8개 구·군별 PM10 / PM2.5 농도 비교 (Chart.js)
- **선 차트** : 하루 시간대별 PM2.5 농도 변화 추이 (Chart.js)
- **등급 안내** : 미세먼지 등급 기준(좋음/보통/나쁨/매우나쁨) 시각적 표시
- **반응형 디자인** : PC / 태블릿(1024px 이하) / 스마트폰(767px 이하) 대응

## 브라우저 실행 방법

별도 서버 설치 없이 브라우저에서 바로 실행할 수 있습니다.

1. `index.html` 파일을 더블클릭
2. 크롬(Chrome) 또는 엣지(Edge) 브라우저에서 열기
3. 인터넷 연결 필요 (Chart.js CDN 로드)
